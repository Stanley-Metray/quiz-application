What kind of database MongoDB is?

  • Graph Oriented
  • Document Oriented
  • Key Value Pair
  • Column Based
MongoDB stores data in JSON structure based documents. These documents in turn contains data in form of key value pairs.

Within how much time does MongDB writes are written to the journal?

  • 60 s
  • 100 ms
  • 1 s
  • 100 s
Writes are physically written to the journal within 100 milliseconds, by default.

Which of the following is not a system collection in MongoDB?

  • database.system.indexes
  • database.system.namespaces
  • admin.system.users
  • admin.system.preferences
There is no system collection to store database preferences.

What is a replica set node which does not maintain its own data but exists only for voting purpose called?

  • Secondary
  • Arbiter
  • Delayed
  • Hidden
We may add an extra mongod instance to a replica set as an arbiter. Arbiters do not maintain a data set. Arbiters only exist to vote in elections. If your replica set has an even number of members, add an arbiter to obtain a majority of votes in an election for primary. Arbiters do not require dedicated hardware

Which format/standard is used by MongoDB internally to store documents?

  • BSON
  • JSON
  • JSON - Extended
  • B+ Trees
MongoDB uses BSON, a binary object format similar to, but more expressive than JSON.
